# **EduWeb**

## **Overview**
Welcome to the Front-End Educational Website repository! This project is designed to provide a user-friendly and visually appealing platform for learners to access educational content.

Features
Homepage: A beautifully crafted landing page that serves as the gateway to a world of knowledge.
Intuitive Design: Navigating through the website is seamless, ensuring a delightful user experience.

## **Getting Started**
To run the project locally, follow these steps:-

Clone the repository
Open index.html in your preferred web browser.
